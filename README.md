# Mentoria Ruby on Rails — Bárbara Garcez

Repositório de acompanhamento da mentoria individual em desenvolvimento de software com foco em **Ruby** e **Ruby on Rails**.

## Objetivo

Capacitar a mentorada ao nível de **Engenheira de Software Júnior**, desenvolvendo habilidades técnicas e de processo para atuar profissionalmente na área.

## Estrutura

- **Duração:** 20 semanas / 3 módulos sequenciais
- **Sessões:** 40 encontros ao vivo de 1h30, 2x por semana
- **Início:** 23/06/2026

## Conteúdo Programático

- Lógica de programação e fundamentos de Ruby
- Desenvolvimento de aplicações web com Ruby on Rails
- Versionamento de código com Git e GitHub
- Testes automatizados
- Integração contínua e publicação em produção
